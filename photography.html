Photography.

This project was to go back to the basics of black and white.
Capturing images in a way that an object is unrecognizeable allows for the image to be more interest. Instead of flipping past, the viewer will spend more time on the image trying to understand what is being shown.
Photography is used not only to capture an image but to support a designers end goals. I have used photography to add texture to a dedsign and even as another element. Photography adds so much to a design, one should not forget to use it.
